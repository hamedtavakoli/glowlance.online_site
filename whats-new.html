What's New

Updates and release notes for VocabGlow.

This page tracks improvements and updates so it is easy to see what changed across versions.

Version 2.5.6

May 4, 2026

Latest

A major usability update focused on categories, popup customisation, saved-word statistics, highlight controls, and clearer settings.

  • Added category support for saved words.
  • Added category dashboard details and category filtering in the full vocabulary list.
  • Added category data to CSV import and export.
  • Added configurable lookup popup content.
  • Added a detailed dictionary source setting for Cambridge Dictionary, Oxford Learner's Dictionaries, or Merriam-Webster.
  • Added highlight style and highlight limit settings.
  • Added popup saved-word stats.
  • Improved settings organisation across lookup, highlighting, reminders, data, privacy, sources, and services.

Version 2.5.3

May 1, 2026

An accessibility, translation-control, and highlighting stability update.

  • Added an online translation toggle so English dictionary lookups can run without translation enabled.
  • Added migration support for existing translation settings and defaulted new installs to translation off.
  • Added lookup loading feedback and temporarily disabled popup actions while lookup results are still loading.
  • Improved popup accessibility with dialog labels, focus restoration, Escape-to-close, focus trapping in the edit modal, and keyboard repositioning.
  • Improved pronunciation playback by normalising audio URLs and falling back between US and UK audio when one dialect is missing audio.
  • Improved saved-word highlighting performance and safety by debouncing refreshes, limiting highlights to recent saved terms, and skipping form, code, button, and editable content.

Version 2.5.2

April 26, 2026

A polish update focused on version visibility, support links, and saved-word management.

  • Added automatic version labels in the popup, options page, and full vocabulary list.
  • Improved support links in the popup and full vocabulary list.
  • Improved the in-page popup styling and saved-word indication.
  • Kept the full-list page focused on search, editing, manual lookup, pronunciation, CSV import/export, and review.

Version 2.5.0

April 24, 2026

A bug-fix update for saved words and highlighting after Version 2.4.0.

  • Fixed an issue where saved words were missing from the full list.
  • Fixed highlighting of saved words after upgrading from Version 2.4.0.

Version 2.4.0

April 24, 2026

A major feature update introducing richer lookup, translation, flexible saving, and improved vocabulary management.

  • Added translation support in more than 100 languages.
  • Added flexible save actions for full meanings, summaries, translations, and edited entries.
  • Improved the in-page popup with richer pronunciation and dictionary details.
  • Improved the full-page vocabulary manager.
  • Improved settings, reminders, and saved-word highlighting.
  • Moved saved vocabulary words to local browser storage for better capacity and performance.
