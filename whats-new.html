What's New

Updates and release notes for VocabGlow.

This page tracks improvements and updates so it is easy to see what changed across versions.

Version 2.5.6

May 4, 2026

Latest

Major update introducing categories, advanced organisation, and flexible customisation for vocabulary management.

  • Added categories for saved words to organise vocabulary by topic or theme
  • Introduced category dashboard and filtering for better vocabulary management
  • Implemented CSV import/export with full category data support
  • Added configurable lookup popup content options
  • Added dictionary source setting with Cambridge, Oxford Learner's Dictionaries, and Merriam-Webster
  • Added highlight style and highlight limit settings for custom highlighting
  • Added popup saved-word statistics for tracking progress
  • Improved settings organisation for better usability
